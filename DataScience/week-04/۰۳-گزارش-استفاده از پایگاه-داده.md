# یادداشت‌ها و مراحل کار

## نصب psycopg2

کتاب می‌خواهد از psycopg2 استفاده کند، که به نظر می‌رسد با دستوری که در کتاب هست مستقیم نصب نمی‌شود و باید از دستور دیگری که آورده‌ام استفاده کرد.
```bash
pip install psycopg2#دستوری که در کتاب نوشته، به نظر این برای چیزی شبیه کامپایل است
pip install psycopg2-binary # دستوری که با آن نصب می‌شود.
```
به نظر نسخه‌ی سوم این ماژول نیز وجود دارد که ویژگی‌های جدید در آن منتشر می‌شوند اما مطابق کتاب و به این دلیل که منسوخ نشده بود از همان نسخه دو استفاده کردم.
